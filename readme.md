# Cluster / Worker Threads Example #

## Install
* Clone project:
```bash 
$ git clone <project-url>
```
* Install dependency package:
```bash
$ npm install
```

## Build and Run
### Cluster
* Run system (use node):
```bash
$ node cluster.js
```
* Run system (use npm):
```bash
$ npm run cluster
```

### Worker Threads
* Run system (use node):
```bash
$ node worker_threads.js
```
* Run system (use npm):
```bash
$ npm run worker_threads
```
